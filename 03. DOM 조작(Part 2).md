# 1. 요소 노드 생성과 추가 ⭐️

<details>
  <summary>(1) createElement</summary>

  * **요소 생성**
  ```JavaScript
  // 요소 생성
  const div = document.createElement('div');
  const p = document.createElement('p');
  const button = document.createElement('button');

  console.log(div)     // <div></div>
  console.log(p);      //<p></p>
  console.log(button); //<button></button>

  // 아직 화면에 표시되지 않음
  ```

  * **내용과 속성 추가**
  ```HTML
  <h1>기존 제목</h1>
  ```
  ```JavaScript
  const heading = document.createElement('h1');

  // 텍스트 추가
  heading.textContent = '새로운 제목';

  // 클래스 추가
  heading.className = 'title';
  // 또는
  heading.classList.add('title', 'main');
  // ID 추가
  heading.id = 'main-heading';

  // 스타일 추가
  heading.style.color = 'blue';
  heading.style.fontSize = '24px';

  // 속성 추가 (주로 커스텀 속성에만 쓸 것 권장)
  heading.setAttribute('data-id', '123');

  console.log(heading);
  // <h1 id="main-heading" class="title main" data-id="123" style="color: blue; font-size: 24px;">새로운 제목</h1>
  ```
</details>

<details>
  <summary>(2) appendChild</summary>

  * **마지막 자식으로 추가**
  ```HTML
  <div id="container"></div>
  ```
  ```JavaScript
  const container = document.querySelector('#container');
  const newDiv = document.createElement('div');
  newDiv.textContent = '새 요소';

  // container의 마지막 자식으로 추가
  container.appendChild(newDiv);
  ```

  * **여러 요소 추가**
  ```HTML
  <ul>
    <li>항목 0</li>
  </ul>
  ```
  ```JavaScript
  const list = document.querySelector('ul');

  // 5개 항목 추가
  for (let i = 1; i <= 5; i++) {
      const li = document.createElement('li');
      li.textContent = `항목 ${i}`;
      list.appendChild(li);
  }
  ```

  * **이미 DOM에 있는 요소는 이동됨**
  ```JavaScript
  const div1 = document.querySelector('#div1');
  const div2 = document.querySelector('#div2');
  conste element = document.querySelector('.element');

  div1.appendChild(element);  // div1로 이동
  div2.appendChild(element);  // div2로 이동(div1에서 사라짐)
  ```
</details>

<details>
  <summary>(3) append(현대적 방법)</summary>

  * **여러 개 동시 추가**
  ```HTML
  <div id="container"></div>
  ```
  ```JavaScript
  const container = document.querySelector('#container');

  const div1 = document.createElement('div');
  const div2 = document.createElement('div');
  const text = '텍스트도 가능';

  // 여러 개 동시 추가
  container.append(div1, div2, text);

  // appendChild vs append
  container.appendChild(div1);        // 요소만, 반환값 있음
  container.append(div1, div2, text); // 여러 개, 텍스트, 반환값 없음
  ```
</details>

<details>
  <summary>(4) prepend</summary>

  * **첫 번째 자식으로 추가**
  ```HTML
  <ul>
    <li>항목 0</li>
  </ul>
  ```
  ```JavaScript
  const list = document.querySelector('ul');
  const newItem = document.createElement('li');
  newItem.textContent = '맨 앞 항목';

  list.prepend(newItem);

  // 여러 개 가능
  list.prepend(item1, item2, '텍스트');
  ```
</details>

<details>
  <summary>(5) insertBefore</summary>

  * **특정 요소 앞에 삽입**
  ```HTML
  <div id="parent">
    <div id="reference">레퍼런스</div>
  </div>
  ```
  ```JavaScript
  const parent = document.querySelector('#parent');
  const newElement = document.createElement('div');
  const referenceElement = document.querySelector('#reference');

  // referenceElement 앞에 newElement 삽입
  // referenceElement는 반드시 parent의 직속 자식이어야 힘
  parent.insertBefore(newElement, referenceElement);

  // 맨 앞에 추가(첫 번째 자식 앞)
  parent.insertBefore(newElement, parent.firstChild);
  ```
</details>

<details>
  <summary>(6) before / after</summary>

  * **형제로 추가**
  ```HTML
  <div id="parent">
    <div id="element">레퍼런스</div>
  </div>
  ```
  ```JavaScript
  const element = document.querySelector('#element');
  const newBefore = document.createElement('div');
  const newAfter = document.createElement('div');

  // element 앞에
  element.before(newBefore);

  // element 뒤에
  element.after(newAfter);

  // 여러 개 가능
  element.before(el1, el2, '텍스트');
  ```
</details>

# 2. 노드 삭제와 이동 ⭐️

<details>
  <summary>(1) remove</summary>

  * **자신을 삭제**
  ```HTML
  <div id="parent">
    <div id="element">타겟</div>
    <ul>
      <li class="item">리스트 1</li>
      <li class="item">리스트 2</li>
      <li class="item">리스트 3</li>
    </ul>
  </div>
  ```
  ```JavaScript
  const element = document.querySelector('#element');

  // 요소 제거
  element.remove();

  // 여러 요소 제거
  const items = document.querySelectorAll('.item');
  items.forEach(item => item.remove());
  ```

  * **조건부 삭제**
  ```HTML
  <div id="parent">
    <h1>투두리스트</h1>
    <ul>
      <li class="todo">리스트 1</li>
      <li class="todo">리스트 2</li>
      <li class="todo completed">리스트 3</li>
      <li class="todo completed">리스트 4</li>
    </ul>
    <div class="temporary">임시 요소1</div>
    <div class="temporary">임시 요소2</div>
  </div>
  ```
  ```JavaScript
  // 완료된 할 일 삭제
  const completedTodos = document.querySelectorAll('.todo.completed');
  completedTodos.forEach(todo => todo.remove());

  // 특정 클래스 삭제
  document.querySelectorAll('.temporary').forEach(el => el.remove());
  ```
</details>

<details>
  <summary>(2) removeChild</summary>

  * **자식 요소 삭제**
  ```HTML
  <div id="parent">
    <div>자식 1</div>
    <div id="child">자식 2</div>
    <div>자식 3</div>
  </div>
  ```
  ```JavaScript
  const parent = document.querySelector('#parent');
  const child = document.querySelector('#child');

  // 특정 자식 제거
  parent.removeChild(child);

  // 첫 번째 자식 제거
  if (parent.firstElementChild) {
      parent.removeChild(parent.firstElementChild);
  }

  // 마지막 자식 제거
  if (parent.lastElementChild) {
      parent.removeChild(parent.lastElementChild);
  }
  ```

  * **모든 자식 제거**
  ```HTML
  <div id="container">
    <div>자식 1</div>
    <div>자식 2</div>
    <div>자식 3</div>
  </div>
  ```
  ```JavaScript
  const container = document.querySelector('#container');

  // 방법 1: while
  while (container.firstChild) {
      container.removeChild(container.firstChild);
  }

  // 방법 2: innerHTML (빠르지만 이벤트 리스너 제거 안 됨. 메모리 누수현상)
  container.innerHTML = '';

  // 방법 3: replaceChildren(권장)
  container.replaceChildren();
  ```
</details>

<details>
  <summary>(3) replaceChild</summary>

  * **자식 요소 교체**
  ```HTML
  <div id="parent">
    <div id="old">오래된 요소</div>
  </div>
  ```
  ```JavaScript
  const parent = document.querySelector('#parent');
  const oldChild = document.querySelector('#old');
  const newChild = document.createElement('div');
  newChild.textContent = '새 요소';

  // oldChild를 newChild로 교체
  parent.replaceChild(newChild, oldChild);
  ```
</details>

<details>
  <summary>(4) replaceWith</summary>

  * **자신을 다른 요소로 교체**
  ```HTML
  <div id="parent">
    <div id="old">오래된 요소</div>
  </div>
  ```
  ```JavaScript
  const oldElement = document.querySelector('#old');
  const newElement = document.createElement('div');
  newElement.textContent = '새 요소';

  // oldElement를 newElement로 교체
  oldElement.replaceWith(newElement);

  // 여러 개로 교체 가능
  oldElement.replaceWith(el1, el2, '텍스트');
  ```
</details>

<details>
  <summary>(5) 요소 이동</summary>

  * 이미 DOM에 있는 요소를 append 하면 **삽입이 아니라 그냥 이동이 됨**

  ```HTML
  <div id="parent">
    <div id="element">자식요소</div>
  </div>
  <div id="new-parent">
  </div>
  ```
  ```JavaScript
  // 이동은 append로!
  const element = document.querySelector('#element');
  const newParent = document.querySelector('#new-parent');

  // element가 newParent의 자식으로 이동
  newParent.append(element);
  ```
</details>

# 3. HTML 속성 다루기 ⭐️

<details>
  <summary>(1) getAttribute / setAttribute</summary>

  ```HTML
  <img id="myImg" src="image.jpg" alt="이미지">
  <a id="myLink" href="https://example.com">링크</a>
  ```
  ```JavaScript
  const img = document.querySelector('#myImg');
  const link = document.querySelector('#myLink');

  // 읽기
  console.log(img.getAttribute('src'));  // "image.jpg"
  console.log(img.getAttribute('alt'));  // "이미지"

  // 쓰기
  img.setAttribute('src', 'new-image.jpg');
  img.setAttribute('alt', '새 이미지');

  // 없는 속성도 추가 가능
  img.setAttribute('data-id', '123');

  // 링크 변경
  link.setAttribute('href', 'https://google.com');
  link.setAttribute('target', '_blank');
  ```
</details>

<details>
  <summary>(2) 직접 속성 접근</summary>

  ```HTML
  <img id="myImg" src="https://picsum.photos/100" alt="원본 이미지">
  <input type="checkbox" value="초기값">
  ```
  ```JavaScript
  const img = document.querySelector('#myImg');

  // 표준 속성은 프로퍼티로 접근 가능
  console.log(img.src);  // 절대 경로
  console.log(img.alt);

  // 쓰기
  img.src = 'new-image.jpg';
  img.alt = '새 이미지';

  // input 요소
  const input = document.querySelector('input');
  console.log(input.value);
  console.log(input.type);
  console.log(input.checked);  // checkbox/radio
  console.log(input.disabled);

  input.value = '새 값';
  input.disabled = true;
  ```
</details>

<details>
  <summary>(3) hasAttribute / removeAttribute</summary>

  ```HTML
  <a href="https://google.com" target="_blank">구글 링크</a>
  <button disabled>버튼</button>
  ```
  ```JavaScript
  const link = document.querySelector('a');

  // 속성 존재 확인
  if (link.hasAttribute('target')) {
      console.log('target 속성이 있음');
  }

  // 속성 제거
  link.removeAttribute('target');

  // disabled 토글
  const button = document.querySelector('button');

  if (button.hasAttribute('disabled')) {
      button.removeAttribute('disabled');
  } else {
      button.setAttribute('disabled', '');
  }

  // 또는 간단하게
  button.disabled = !button.disabled;
  ```
</details>

<details>
  <summary>(4) className vs classList</summary>

  ```HTML
  <div class="box active old">테스트 요소</div>
  ```
  ```JavaScript
  const element = document.querySelector('.box');

  // className
  console.log(element.className);   // "box active"
  element.className = 'new-class';
  element.className += ' another';

  // classList - 권장!
  element.classList.add('highlight');
  element.classList.remove('active');
  element.classList.toggle('selected');
  element.classList.contains('box');  // true
  element.classList.replace('old', 'new');
  ```
</details>

# 5. 비표준 속성 다루기(data-*) ⭐️

<details>
  <summary>(1) data 속성이란?</summary>

  * 개발자가 **나중에 활용할 목적**으로 HTML 요소에 추가적인 정보를 저장하는 속성
  ```HTML
  <div
      id="user"
      data-id="123"
      data-name="홍길동"
      data-role="admin"
      data-created-at="2024-01-20"
  >
      사용자 정보
  </div>
  ```

  * **특징**
    * `data-`로 시작
    * 사용자 정의 데이터 저장
    * HTML 표준
    * CSS에서도 사용 가능
</details>

<details>
  <summary>(2) dataset API</summary>

  * **data-** 로 저장한 데이터에 접근하는 방법
  ```JavaScript
  const user = document.querySelector('#user');

  // 읽기 (카멜 케이스로 변환됨)
  console.log(user.dataset.id);       // "123"
  console.log(user.dataset.name);     // "홍길동"
  console.log(user.dataset.role);     // "admin"
  console.log(user.dataset.createAt); // "2024-01-20"

  // 쓰기
  user.dataset.status = 'active';
  user.dataset.lastLogin = new Date().toISOString();

  // 삭제
  delete user.dataset.status;

  // 전체 data 객체
  console.log(user.dataset);
  // DOMStringMap {id: "123", name: "홍길동",...}
  ```

  * **클릭한 요소가 어떤 데이터인지 알아야할 때 유용**
  ```HTML
  <div class="product" data-id="1" data-price="10000">아이폰</div>
  <div class="product" data-id="2" data-price="20000">갤럭시</div>
  <div class="product" data-id="3" data-price="30000">픽셀</div>
  ```
  ```JavaScript
  document.querySelectorAll('.product').forEach(el => {
      el.addEventListener('click', (e) => {
          console.log(e.target.dataset.id)      // "1"
          console.log(e.target.dataset.price)   // "10000"
          // 이 데이터로 장바구니에 담기, 서버에 요청 등
  ```
</details>

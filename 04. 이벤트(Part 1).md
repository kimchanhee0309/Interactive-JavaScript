# 1. 이벤트와 이벤트 핸들러⭐️

<details>
  <summary>(1) 이벤트란?</summary>

  ```JavaScript
  이벤트 = 웹 페이지에서 발생하는 모든 사건

  예:
  - 사용자가 버튼을 클릭
  - 키보드로 입력
  - 마우스를 움직임
  - 페이지가 로드됨
  - 스크롤함
  ```

  * **주요 이벤트 종류**
  ```JavaScript
  // 마우스 이벤트
  click       // 클릭 ⭐️
  dblclick    // 더블 클릭
  mousedown   // 마우스 버튼 누름
  mouseup     // 마우스 버튼 뗌
  mousemove   // 마우스 이동
  mouseenter  // 마우스 진입
  mouseleave  // 마우스 이탈

  // 키보드 이벤트
  keydown     // 키 누름
  keyup       // 키 뗌
  keypress    // 키 입력 (deprecated)

  // 폼 이벤트
  submit      // 폼 제출 ⭐️
  input       // 입력값 변경
  change      // 값 변경 완료 ⭐️
  focus       // 포커스 받음
  blur        // 포커스 잃음

  // 문서 이벤트
  DOMContentLoaded  // DOM 로드 완료
  load              // 모든 리소스 로드 완료
  scroll            // 스크롤
  resize            // 창 크기 변경
  ```
</details>

<details>
  <summary>(2) 이벤트 핸들러 등록 방법</summary>

  * **이벤트 핸들러** : 어떤 이벤트가 발생했을 때, 그로인해 처리해야 할 내용이 생기는데, 이 처리를 담당하는 함수
  * **방법1: HTML 속성(비권장)**
  ```JavaScript
  <button onclick="alert('클릭!')">클릭</button>
  <button onclick="handleClick()">클릭</button>

  <script>
  function handleClick() {
      alert('클릭됨!');
  }
  </script>
  ```

  * **방법2: DOM 프로퍼티**
  ```JavaScript
  const button = document.querySelector('button');

  button.onclick = function() {
      alert('클릭됨!');
  };

  // 단점: 하나만 등록 가능(덮어씀)
  button.onclick = function() {
      alert('첫 번째');
  };
  button.onclick = function() {
      alert('두 번째');  // 이것만 실행됨
  };
  ```

  * **방법3: addEventListener(권장)**
  ```JavaScript
  const button = document.querySelector('button');

  button.addEventListener('click', function() {
      alert('클릭됨!');
  });

  // 여러 개 등록 가능
  button.addEventListener('click', function() {
      console.log('첫 번째');
  });
  button.addEventListener('click', function() {
      console.log('두 번째');
  });
  // 둘 다 실행
  ```

</details>

<details>
  <summary>(3) addEventListener 사용법</summary>

  ```JavaScript
  element.addEventListener(이벤트타입, 핸들러함수, 옵션);

  // 기본 사용
  button.addEventListener('click', function() {
      console.log('클릭툄');
  });

  // 화살표 함수
  button.addEventListener('click', () => {
      console.log('클릭툄');
  });

  // 함수 분리
  function handleClick() {
      console.log('클릭됨');
  }
  button.addEventListener('click', handleClick);

  // 매개변수 전달(화살표 함수로 감싸기)
  function greet(name) {
      console.log(`안녕하세요, ${name}님!`);
  }
  button.addEventListener('click', () => greet('홍길동'));
  ```
</details>

<details>
  <summary>(4) removeEventListener</summary>

  * **이벤트 핸들러 제거**
  ```JavaScript
  function handleClick() {
      console.log('클릭됨');
  }

  // 등록
  button.addEventListener('click', handleClick);

  // 제거
  button.removeEventListener('click', handleClick);

  // ❌ 익명 함수는 제거 불가
  button.addEventListener('click', function() {
      console.log('클릭');
  });

  button.removeEventListener('click', function() {
      console.log('클릭');
  });
  // 제거 안 됨!(다른 함수로 인식)
  ```

  * **한 번만 실행**
  ```JavaScript
  // 방법 1: once 옵션(권장)
  button.addEventListener('click', function() {
      console.log('한 번만 실행');
  }, { once: true });

  // 방법 2: 직접 제거
  function handleClick() {
      console.log('한 번만 실행');
      button.removeEventListener('click', handleClick);
  }
  button.addEventListener('click', handleClick);
  ```

</details>

# 2. 이벤트 객체⭐️

<details>
  <summary>(1) 이벤트 객체란?</summary>

  * **이벤트 객체** : 이벤트 발생 시 전달되는 객체
  ```JavaScript
  button.addEventListener('click', function(event) {
      console.log(event);  // 이벤트 객체
  });

  // event, e, evt 등으로 받음
  button.addEventListener('click', function(e) {
      console.log(e);
  });

  // 화살표 함수
  button.addEventListener('click', (e) => {
      console.log(e);
  });
  ```
</details>

<details>
  <summary>(2) 이벤트 객체 속성</summary>

  * 주로 쓰이는 event 객체 속성 : **e.target**, **e.currentTarget**
  ```JavaScript
  button.addEventListener('click', function(e) {
      // 이벤트 타입
      console.log(e.type);          // "click"

      // 이벤트 대상(이벤트가 발생한 요소)
      console.log(e.target);        // 클릭된 요소

      // 이벤트 리스너가 등록된 요소
      console.log(e.currentTarget);  // button

      // 타임스탬프
      console.log(e.timeStamp);      //1234567890.123

      // 기본 동작 방지 여부
      console.log(e.defaultPrevented);  // false
  ```
</details>

<details>
  <summary>(3) 마우스 이벤트 속성</summary>

  ```JavaScript
  element.addEventListener('click', function(e) {
      // 마우스 좌표(뷰포트 기준)
      console.log(e.clientX, e.clientY); ⭐️

      // 마우스 좌표(페이지 기준)
      console.log(e.pageX, e.pageY); ⭐️

      // 마우스 좌표(화면 기준)
      console.log(e.screenX, e.screenY);

      // 마우스 버튼
      console.log(e.button);  // 0: 왼쪽, 1: 휠, 2: 오른쪽

      // 수정 키
      console.log(e.ctrlKey);    // Ctrl 눌림 여부
      console.log(e.shiftKey);   // Shift 눌림 여부
      console.log(e.altKey);     // Alt 눌림 여부
      console.log(e.metaKey);    // Cmd(Mac) / Win 키
  });
  ```
</details>

<details>
  <summary>(4) 키보드 이벤트 속성</summary>

  ```JavaScript
  input.addEventListener('keydown', function(e) {
      // 눌린 키
      console.log(e.Key); ⭐️    // "a", "Enter", "ArrowUp" 등
      console.log(e.conde);     // "KeyA", "Enter", "ArrowUp" 등

      // 키 코드(deprecated)
      console.log(e.KeyCode);    // 65 (a), 13 (Enter) 등

      // 수정 키
      console.log(e.ctrlKey);
      console.log(e.shiftKey);
      console.log(e.altKey);
  ```
</details>

<details>
  <summary>(5) preventDefault()⭐️</summary>

  * **기본 동작 방지**
  ```JavaScript
  // 링크 클릭 방지
  const link = document.querySelector('a');
  link.addEventListener('click', function(e) {
      e.preventDefault();
      console.log('링크 이동 방지됨');
  });

  // 폼 제출 방지 ⭐️
  const form = document.querySelector('form');
  form.addEventListener('submit', function(e) {
      e.preventDefault();
      console.log('폼 제출 방지됨');

      // 직접 처리
      const formData = new FormData(form);
      console.log('폼 데이터: ', Object.fromEntries(formData));
  });

  // 우클릭 방지
  document.addEventListener('contextmenu', function(e) {
      e.preventDefault();
      console.log('우클릭 방지됨');
  });
  ```
</details>

# 1. 웹 서비스와 JavaScript

<details>
  <summary>(1) 웹의 3요소</summary>

  ```JavaScript
  HTML (구조)
  - 집의 뼈대
  - 제목, 문단, 버튼 등

  CSS (디자인)
  - 인테리어
  - 색상, 크기, 위치 등

  JavaScript (동작)
  - 전기, 수도 등 기능
  - 클릭, 애니메이션, 데이터 처리 등
  ```

  * **HTML만**
  ```HTML
  <h1>안녕하세요</h1>
  <button>클릭</button>

  // → 버튼은 있지만 클릭해도 아무 일도 안 일어남
  ```

  * **HTML + JavaScript**
  ```HTML
  <h1>안녕하세요</h1>
  <button onClick="alert('안녕!)">클릭</button>

  // → 버튼 클릭 시 알림창 표시!
  ```
</details>

<details>
  <summary>(2) JavaScript로 할 수 있는 것</summary>

  * **웹 페이지에서**
  ```JavaScript
  // 1. 내용 변경
  document.querySelector('h1').textContent = '새 제목';

  // 2. 스타일 변경
  document.querySelector('button').style.color = 'red';

  // 3. 요소 추가/삭제
  const newDiv = docunemt.createElement("div");
  newDiv.textContent = "새로운 내용";
  document.body.appendChild(newDiv);

  // 4. 이벤트 처리
  document.querySelector('button').addEventListener('click', () => {
      alert('클릭됨!');
  });

  // 5. 애니메이션
  const h1 = document.querySelector("h1");
  h1.style.transition = "transform ls ease";
  h1.style.transform = "translateX(100px)";

  // 6. 데이터 통신
  fetch('https://jsonplaceholder.typicode.com/todos/1')
      .then(response => response.json())
      .then(json => console.log(json))
  ```
</details>

# 1. 웹 서비스와 JavaScript

<details>
  <summary>(1) 웹의 3요소</summary>

  ```JavaScript
  HTML (구조)
  - 집의 뼈대
  - 제목, 문단, 버튼 등

  CSS (디자인)
  - 인테리어
  - 색상, 크기, 위치 등

  JavaScript (동작)
  - 전기, 수도 등 기능
  - 클릭, 애니메이션, 데이터 처리 등
  ```

  * **HTML만**
  ```HTML
  <h1>안녕하세요</h1>
  <button>클릭</button>

  // → 버튼은 있지만 클릭해도 아무 일도 안 일어남
  ```

  * **HTML + JavaScript**
  ```HTML
  <h1>안녕하세요</h1>
  <button onClick="alert('안녕!)">클릭</button>

  // → 버튼 클릭 시 알림창 표시!
  ```
</details>

<details>
  <summary>(3) 웹 브라우저의 역할</summary>

  ```JavaSCript
  사용자가 웹 페이지 접속
         ↓
  브라우저가 HTML, CSS, JS 다운로드
         ↓
  HTML 파싱 → DOM 생성
  CSS 파싱 → 스타일 적용
  JS 실행 → DOM 조작
         ↓  
  화면에 렌더링
  ```
  * **브라우저가 제공하는 것**
  ```JavaSCript
  // Window 객체(브라우저 창)
  window.innerWidth;  // 창 너비
  window.location;    // 현재 URL
  window.alert();     // 알림창

  // Document 객체(HTML 문서)
  document.title;            // 페이지 제목
  document.querySelector();  // 요소 선택
  document.createElement();  // 요소 생성

  // Console 객체(개발자 도구)
  console.log();    // 로그 출력
  console.error();  // 에러 출력
  ```

<details>
  <summary>(4) 일반적인 실수</summary>

  * **❌ JavaScript가 HTML보다 먼저 실행**
  ```HTML
  <head>
      <script>
          // 이 시점에 body가 아직 없음!
          const title = document.getElementById('title');
          console.log(title);  // null
      </script>
  </head>
  <body>
      <h1 id="title">제목</h1>
  </body>
  ```

  * **✅ 해결 방법 1 - body 끝에 배치**
  ```HTML
  <body>
      <h1 id="title">제목</h1>

      <script>
          // 이제 title이 존재함
          const title = document.getElementById('title');
          console.log(title);  // <h1 id="title">제목</h1>
      </script>
  </body>
  ```

  * **✅ 해결 방법 2 - DOMContentLoaded 이벤트**
  ```HTML
  <head>
      <script>
          document.addEventListener('DOMContentLoaded', function() {
              // DOM이 준비된 후 실행
              const title = document.getElementById('title');
              console.log(title); // <h1 id="title">제목</h1>
          });
      </script>
  </head>
  <body>
      <h1 id="title">제목</h1>
  </body>
  ```

  * **✅ 해결 방법 3(권장) - defer 속성**
  ```HTML
  <head>
      <!-- defer: HTML 파싱 후 실행 -->
      <script defer src="script.js"></script>
  </head>
  <body>
      <h1 id="title">제목</h1>
  </body>
  ```
</details>

<details>
  <summary>(2) getElementsByClassName</summary>

  * **클래스로 요소 선택**
  ```HTML
  <div class="box">박스 1</div>
  <div class="box">박스 2</div>
  <div class="box highlight">박스 3</div>
  <p class="description">설명</p>
  ```
  ```JavaScript
  // 클래스로 선택 (HTMLCollection 반환)
  const boxes = document.getElementsByClassName('box');

  console.log(boxes);      // HTMLCollection(3)
  console.log(boxes[0]);   // 첫 번째 box
  console.log(boxes[1]);   // 두 번째 box
  console.log(boxes[2]);   // 세 번째 box

  // 여러 클래스 (둘 다 가진 요소만)
  const highlighted = document.getElementByClassName('box highlight');
  console.log(highlighted.length);  // 1
  ```
  * **순회하기**
  ```JavaScript
  const boxes = document.getElementsByClassName('box');

  // for문
  for (let i = 0; i < boxes.length; i++) {
      console.log(boxes[i].textContent);
  }

  // for...of (동작함)
  for (const box of boxes) {
      console.log(box.textContent);
  }

  // forEach는 안 됨! (HTMLCollection은 배열 아님)
  // boxes.forEach(box => console.log(box));
  // 에러

  // 배열로 변환 후 forEach
  Array.from(boxes).forEach(box => {
      console.log(box.textcontent);
  });

  // 또는
  [...boxes].forEach(box => {
      console.log(box.textContent);
  });
  ```
</details>

# 4. 태그 이름과 CSS 선택자로 선택하기

  ```JavaScript
  const container = document.getElementById("container");
  ```

  ```JavaScript
  const subtitle = document.querySelector('#header p');

  // 자식 선택자

  // 속성 선택자
  const acriveItem = document.querySelector('li.active');

  ```
  * **다양한 CSS 선택자**
  ```JavaScript
  // 여러 클래스
  document.querySelector('.box.highlight');
  // 두 클래스 모두 있는 요소

  // 형제 선택자
  ```

<details>
  <summary>(4) 메서드 비교</summary>

  |메서드|반환값|반환 개수|forEach|Live| 
  |-----|-----|-----|-----|-----|
  |getElementById|Element|1개|-|-|
  |getElementsByClassName|HTMLCollection|여러 개|x|o|
  |getElementsByTagName|HTMLCollection|여러 개|x|o|
  |querySelector|Element|1개(첫 번째)|-|-|
  |querySelectorAll|NodeList|여러 개|o|x|

  * **Live vs Static**
    * `live` = **항상 현재 DOM 반영**, `static` = **호출한 순간 DOM 스냅샷**
    ```JavaScript
    // Live (HTMLCollection)
    const boxes1 = document.getElementsByClassName('box');
    console.log(boxes1.length);  // 3

    document.body.innerHTML += '<div class="box">새 박스</div>';
    console.log(boxes1.length);  // 4(자동 업데이트)

    // Static (NodeList)
    const boxes2 = document.querySelectorAll('.box');
    console.log(boxes2.length);  // 4

    document.body.innerHTML += '<div class="box">또 새 박스</div>';
    console.log(boxes2.length);  // 4(변하지 않음)
    ```
  * **권장 사항**
  ```JavaScript
  // ✅ 현대적 방법(권장)
  const element = document.querySelector('#myId');
  const elements = document.querySelectorAll('.myClass');

  // ✅ 간단한 경우(빠름)
  const byId = document.getElementById('myId');

  // ❌ 복잡한 선택에는 비효율적
  const byClass = document.getElementsByClassName('myClass');
  ```
</details>

# 5. 유사 배열(Array-Like Object)⭐️

<details>
  <summary>(1) 유사 배열이란?</summary>

  * **배열처럼 생겼지만 배열이 아님**
  ```JavaScript
  const boxes = document.querySelectorAll('.box');

  console.log(boxes);          // NodeList(3)
  console.log(boxes[0]);       // 인덱스 접근 가능
  console.log(boxes.length);   // length 있음

  // But..
  console.log(Array.isArray(boxes));  // false

  // 일부 배열 메서드만 가능
  boxes.forEach(box => console.log(box));
  // 가능
  // boxes.map(box => box.textContent);
  // 에러(NodeList는 map 없음)
  ```
</details>

<details>
  <summary>(2) HTMLCollection</summary>

  * **getElementsBy 계열이 반환**
  ```JavaScript
  const boxes = document.getElementsByClassName('box');

  console.log(boxes);          // HTMLCollection(3)
  console.log(boxes.length);   // 3
  console.log(boxes[0]);       // 첫 번째 요소
  console.log(boxes.item(0));  // 첫 번째 요소(동일)

  // forEach 없음!
  // boxes.forEach(box => console.log(box)); 에러

  // for문 사용
  for (let i = 0; i < boxes.length; i++) {
      console.log(boxes[i]);
  }

  // for...of 사용
  for (const box of boxes) {
      console.log(box);
  }

  // 배열로 변환 후 forEach
  Array.from(boxes).forEach(box => {
      console.log(box);
  });
  ```

  * **Live Collection(동적)**
  ```JavaScript
  const boxes = document.getElementsByClassName('box');
  console.log(boxes.length);  // 3

  // 새 요소 추가
  const newBox = document.createElement('div');
  newBox.className = 'box';
  document.body.appendChild(newBox);

  console.log(boxes.length);  // 4(자동 업데이트!)

  // 요소 제거
  boxes[0].remove();
  console.log(boxes.length);  // 3(자동 감소)
  ```
</details>

<details>
  <summary>(3) NodeList</summary>

  * **querySelectorAll이 반환**
  ```JavaScript
  const boxes = document.querySelectorAll('.box');

  console.log(boxes);            // NodeList(3)
  console.log(boxes.length);     // 3
  console.log(boxes[0]);         // 첫 번째 요소

  // forEach 있음
  boxes.forEach((box, index) => {
      console.log(index, box);
  });

  // entries, keys, values
  for (const [index, box] of boxes.entries()) {
      console.log(index, box);
  }
  ```
  * **Static Collection(정적)**
  ```JavaScript
  const boxes = document.querySelectorAll('.box');
  console.log(boxes.length);  // 3

  // 새 요소 추가
  const newBox = document.createElement('div');
  newBox.className = 'box';
  document.body.appendChild(newBox);

  console.log(boxes.length);  // 3(변하지 않음)

  // 다시 선택해야 함
  const updatedBoxes = document.querySelectorAll('.box');
  console.log)updatedBoxes.length);  // 4
  ```
</details>

<details>
  <summary>(4) 배열로 변환하기</summary>

  * **방법 1: Array.from()**
  ```JavaScript
  const boxes = document.querySelectorAll('.box');
  const boxArray = Array.from(boxes);

  console.log(Array.isArray(boxArray));  // true

  // 배열 메서드 사용 가능
  const texts = boxArray.map(box => box.textContent);
  const filtered = boxArray.filter(box => box.classList.contains('active'));
  ```

  * **방법 2: Spread 연산자**
  ```JavaScript
  const boxes = document.querySelectorAll('.box');
  const boxArray = [...boxes];

  const texts = boxArray.map(box => box.textContent);
  ```

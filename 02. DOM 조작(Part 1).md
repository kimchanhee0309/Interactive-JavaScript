# 1. 브라우저와 DOM

<details>
  <summary>(1) 브라우저의 렌더링 과정</summary>

  ```JavaScript
  1. HTML 다운로드
     ↓
  2. HTML 파싱 (Parsing)
      → DOM 트리 생성
     ↓
  3. CSS 파싱
      → CSSOM 트리 생성
     ↓  
  4. DOM + CSSOM
      → 렌더 트리 생성
     ↓
  5. 레이아웃 계산
     ↓
  6. 페인팅 (화면에 그리기)
  ```
</details>

<details>
  <summary>(2) DOM이란?</summary>

  * **Document Object Model(문서 객체 모델)**
  ```JavaScript
  HTML 문서를 객체로 표현한 것

  <html>
    <body>
      <h1>제목</h1>
      <p>문단</p>
    </body>
  </html>

  →

  document
    └─ html
        └─ body
            ├─ h1
            └─ p
  ```

  * **자바스크립트로 접근**
  ```JavaScript
  // document 객체
  console.log(document);
  console.log(document.documentElement);      // <html>
  console.log(document.head);                 // <head>
  console.log(document.body);                 // <body>

  // 정보 가져오기
  console.log(document.title);                // 페이지 제목
  console.log(document.URL);                  // 현재 URL
  console.log(document.domain);               // 도메인
  ```
</details>

<details>
  <summary>(3) DOM과 HTML의 차이</summary>

  * **HTML(텍스트)**
  ```HTML
  <div id="Container">
      <p>문단</p>
  </div>
  ```
  * **DOM(객체)**
  ```JavaScript
  // 자바스크립트로 조작 가능한 객체
  const container = document.getElementById('container');

  // 속성 접근
  console.log(container.id);          // "container"
  console.log(container.children);    // HTMLCollection
  console.log(container.innerHTML);   // "<p>문단</p>"

  // 메서드 호출
  container.appendChild(newElement);
  container.remove();
  ```
</details>

# 2. DOM 트리

<details>
  <summary>(1) 노드(Node)의 종류</summary>

  ```JavaScript
  1. Document Node (문서 노드)
     - document 객체
     - DOM 트리의 최상위

  2. Element Node (요소 노드)
     - HTML 태그 (<div>, <p> 등)
     - 가장 자주 사용

  3. Text Node (텍스트 노드)
     - 태그 안의 텍스트
     - <p>여기</p> → "여기"

  4. Comment Node (주석 노드)
     - <!-- 주석 -->

  5. Attribute Node (속성 노드)
     - id, class 등
  ```
  * **예제**
  ```HTML
  <div id="container">
      안녕하세요
      <span>반갑습니다</span>
  </div>

  트리 구조:

  div (Element Node)
   ├─ id="container" (Attribute)
   ├─ "안녕하세요" (Text Node)
   └─ span (Element Node)
       └─ "반갑습니다" (Text Node)
  ```
</details>

<details>
  <summary>(2) 노드 타입 확인</summary>

  ```JavaScript
  const div = document.querySelector('div');

  // nodeType(숫자)
  console.log(div.nodeType);            // 1(Element)
  console.log(div.firstChild.nodeType); // 3(Text)

  // nodeName
  console.log(div.nodeName);  // "DIV"
  console.log(div.firstChild.nodeName);  //"#text"

  // nodeType 상수
  console.log(Node.ELEMENT_NODE);    // 1
  console.log(Node.TEXT_NODE);       // 3
  console.log(Node.COMMENT_NODE);    // 8
  console.log(Node.DOCUMENT_NODE);   // 9
  ```
</details>

<details>
  <summary>(3) 요소 노드 vs 텍스트 노</summary>

  ```HTML
  <p>안녕하세요</p>
  ```
  ```JavaScript
  const p = document.querySelector('p');

  // 요소 노드
  console.log(p);          // <p>안녕하세요</p>
  console.log(p.nodeType); // 1
  console.log(p.nodeName); // "P"

  // 텍스트 노드
  console.log(p.firstChild);           // "안녕하세요"
  console.log(p.firstChild.nodeType);  // 3
  console.log(p.firstChild.nodename);  // "#text"
  console.log(p.firstChild.nodeValue); // "안녕하세요"
  ```
</details>

# 3. DOM 트리 탐색 ⭐️

<details>
  <summary>(1) 부모 노드</summary>

  ```HTML
  <div id="parent">
      <p id="child">자식</p>
  </div>
  ```
  ```JavaScript
  const child = document.getElementById('child');

  // 부모 요소 노드
  const parent =  child.parentElement;
  console.log(parent);          // <div id="parent">
  console.log(parent.id);       // "parent"

  // 부모 노드(같은 결과, 하지만 Node 반환)
  const parentNode = child.parentNode;
  console.log(parentNode);      // <div id="parent">

  // 차이점: document의 부모
  console.log(document.documentElement.parentElement);  // null
  console.log(document.documentElement.parentNode);     // document
  ```

  * **조상 찾기 - closest()**
  ```HTML
  <div class="container">
      <div class="box">
          <p>
              <span id="target">티켓</span>
          </p>
      </div>
  </div>
  ```
  ```JavaScript
  const target = document.getElementById('target');

  // 가장 가까운 .box 찾기
  const box = target.closest('.box');
  console.log(box);    // <div class="box">

  // 가장 가까운 .container 찾기
  const container = target.closest('.container');
  console.log(contrainer);  // <div class="container">

  // 자기 자신도 포함
  const self = target.closest('#target');
  console.log(self);  // <span id="target">

  // 없으면 null
  const notFound = target.closest('.not-exist');
  console.log(notFound);  // null
  ```
</details>

<details>
  <summary>(2) 자식 노드</summary>

  ```HTML
  <ul id="list">
      <li>항목 1</li>
      <li>항목 2</li>
      <li>항목 3</li>
  </ul>
  ```
  ```JavaScript
  const list = document.getElementById('list');

  // 모든 자식 요소 (HTMLCollection)
  const children = list.children;
  console.log(children);         // HTMLCollection(3)
  console.log(children.length);  // 3
  console.log(children[0]);      // 첫 번째 <li>

  // 첫 번째 자식 요소
  const firstChild = list.firstElementChild;
  console.log(firstChild);  // <li>항목 1</li>

  // 마지막 자식 요소
  const lastChild = list.lastElementChild;
  console.log(lastChild);  // <li>항목 2</li>

  // 자식 개수
  const childCount = list.childElementCount;
  console.log(childCount);  // 3
  ```

  * **주의: childNodes vs children**
  ```HTML
  <div id="container">
      안녕
      <span>하세요</span>
  </div>
  ```
  ```JavaScript
  const container = document.getelementById('container');

  // childNodes: 텍스트 노드 포함(NodeList)
  console.log(container.childNodes);
  // NodeList(3) [text, span, text]

  // children: 요소 노드만 (HTMLCollection)
  console.log(container.children);
  // HTMLCollection(1) [span]
  ```
</details>

<details>
  <summary>(3) 형제 노드</summary>

  ```HTML
  <div>
      <p id="first">첫 번째</p>
      <p id="second">두 번째</p>
      <p id="third">세 번째</p>
  </div>
  ```
  ```JavaScript
  const second = document.getElementById('second');

  // 이전 형제 요소
  const prev = second.previousElementSibling;
  console.log(prev);     // <p id="first">
  console.log(prev.id);  // "first"

  // 다음 형제 요소
  const next = second.nextElementSibling;
  console.log(next);     // <p id="third">
  console.log(next.id);  // "third"

  // 없으면 null
  const first = document.getElementById('first');
  console.log(first.previousElementSibling);  // null
  ```
</details>

# 4. 요소 노드 프로퍼티(innerHTML, textContent 등) ⭐️

<details>
  <summary>(1) textContent</summary>

  * **순수 텍스트 읽기/쓰기**
  ```HTML
  <p id="text">안녕하세요 <strong>여러분</strong>
  </p>
  ```
  ```JavaScript
  const p = document.getElementById('text');

  // 읽기(HTML 태그 제외, 텍스트만)
  console.log(p.textContent);  // "안녕하세요 여러분"

  // 쓰기(HTML 태그는 문자 그대로)
  p.textContent = '새로운 <strong>텍스트</strong>';  // 결과: 새로운 <strong>텍스트</strong> (태그가 문자로 표시됨)

  // 안전함 (XSS 공격 방지)
  const userInput = '<script>alert("해킹")</script>';
  p.textContent = userInput;  // 스크립트가 실행 안 됨
  ```
</details>

<details>
  <summary>(2) innerHTML</summary>

  * **HTML 포함 읽기/쓰기**
  ```HTML
  <div id="content">
      <h2>제목</h2>
      <p>내용</p>
  </div>
  ```
  ```JavaScript
  const div = document.getElementById('content');

  // 읽기 (HTML 포함)
  console.log(div.innerHTML);
  // "<h2>제목</h2><p>내용</p>"

  // 쓰기 (HTML 파싱됨)
  div.innerHTML = '<strong>굵은</strong> 텍스트';
  // 결과: 굵은 텍스트(굵게 표시됨)

  // 추가
  div.innerHTML += '<p>추가 문단</p>';

  // 복잡한 HTML
  div.innerHTML = `
      <div class="card">
          <h3>카드 제목</h3>
          <p>카드 내용</p>
          <button>버튼</button>
      </div>
  `;
  ```

  * **보안 주의**
  ```JavaScript
  // ❌ 위험: 사용자 입력을 그대로 사용
  const userInput = '<img src=x onerror="alert(\'XSS\')">';
  div.innerHTML = userInput; // 스크립트 실행됨

  // ✅ 안전: textContent 사용
  div.textContent = userInput;  // 문자로 표시됨

  // ✅ 안전: 이스케이프 함수 사용
  function escapeHtml(text) {
      const div = document.createElement('div');
      div.textContent = text;
      return div.innerHTML;
  }

  div.innerHTML = escapeHtml(userInput);
  ```
</details>

<details>
  <summary>(3) innerText</summary>

  * **화면에 보이는 텍스트**
  ```HTML
  <div id="box">
      표시되는 텍스트
      <span style="display: none;">숨김 텍스트</span>
  </div>
  ```
  ```JavaScript
  const box = document.getElementById('box');

  // textContent: 모든 텍스트
  console.log(box.textContent);
  // "표시되는 텍스트 숨김 텍스트"

  // innerText: 보이는 텍스트만
  console.log(box.innerText);
  // "표시되는 텍스트"
  ```

  * **차이점**
    
  |특징|textContent|innerText|
  |-----|-----|-----|
  |숨김 요소|포함|제외|
  |CSS 영향|무시|반영|
  |성능|빠름|느림(리플로우 발생)|
  |권장|O|특별한 경우만|
</details>

<details>
  <summary>(4) outerHTML</summary>

  * **요소 자체를 포함한 HTML**
  ```HTML
  <div id="target">
      <p>내용</p>
  </div>
  ```
  ```JavaScript
  const target = document.getElementById('target');

  // 읽기 (요소 자체 포함)
  console.log(target.outerHTML);
  // '<div id ="target"><p>내용</p></div>'

  // 쓰기 (요소 자체를 교체)
  target.outerHTML = '<section>새 요소</section>';
  // 결과: div가 사라지고 section으로 교체됨

  // 주의: target 변수는 여전히 이전 요소를 참조
  console.log(target);  // 여전히 div (DOM에는 없음)
  ```
</details>

<details>
  <summary>(5) insertAdjacentHTML</summary>

  * **특정 위치에 HTML 삽입**
  ```HTML
  <div id="target">기존 내용</div>
  ```
  ```JavaScript
  const target = document.getElementById('target');

  // beforebegin: 요소 바로 앞
  target.insertAdjacentHTML('beforebegin', '<p>앞</p>');

  // afterbegin: 첫 번째 자식으로
  target.insertAdjacentHTML('afterbegin', '<span>시작</span>');

  // beforeend: 마지막 자식으로
  target.insertAdjacentHTML('beforeend', '<span>끝</span>');

  // afterend: 요소 바로 뒤
  target.insertAdjacentHTML('afterend', '<p>뒤</p>');
  ```
  ```
  결과:

  <p>앞</p>
  <div id="target">
      <span>시작</span>
      기존 내용
      <span>끝</span>
  </div>
  <p>뒤</p>
  ```
</details>
